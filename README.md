# The-graveyard-breathes
This is ual creative making team2 final project **The graveyard breathes**.

It is an interactive underwater memorial where visitors explore a seabed filled with forgotten memories. Through hand gestures, users can uncover traces of memories that have been buried beneath the surface.

The project used MediaPipe.tox which is not included in this repository, please download the file from the "toxes" folder in [plugin repository](https://github.com/torinmb/mediapipe-touchdesigner.git) and put MediaPipe.tox in the "toxes" folder in this repository to run the project.
